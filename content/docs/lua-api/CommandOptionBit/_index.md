# alias CommandOptionBit
---



```lua
(alias) CommandOptionBit = (4|8|16|32|64|128)
    | 4 -- Meta (windows/mac/mod4) key.
    | 8 -- Internal order.
    | 16 -- Right mouse key.
    | 32 -- Shift key.
    | 64 -- Control key.
    | 128 -- Alt key.

```
