# alias CommandOptionName
---



```lua
(alias) CommandOptionName = ("right"|"alt"|"ctrl"|"shift"|"meta")
    | "right" -- Right mouse key.
    | "alt" -- Alt key.
    | "ctrl" -- Control key.
    | "shift" -- Shift key.
    | "meta" -- Meta key (space).

```
